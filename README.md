# SUM26001 Final Project

## Project Description

### Problem Statement
This project explores the factors associated with student academic performance. Academic performance is influenced by many factors, such as study time, attendance, family background, and prior preparation. Understanding these relationships can help identify which factors are most strongly related to student outcomes.

### Objectives
The objectives of this project are:
1. To define a clear research question related to student academic performance.
2. To explore possible factors that may influence academic outcomes.
3. To demonstrate a simple analytical workflow including problem definition, exploratory analysis, and interpretation.
4. To document bottlenecks encountered during the project and how they were addressed.

### Progress Made / Lessons Learned
During this project, I developed a complete project workflow from problem definition to report writing. I designed the structure of the analysis, considered possible variables related to student performance, and organized the findings into a coherent report.

One major lesson learned is that even when a project does not produce highly advanced predictive results, it is still possible to present meaningful analytical progress by clearly describing the goal, methods, limitations, bottlenecks, and adaptations.
